Getting Started

Memex is a personal link collection displayed as an interactive mind map. Deploy with Docker Compose:

cp .env.example .env
nano .env   # set ADMIN_PASSWORD, SECRET_KEY, APP_PORT
docker compose up -d --build

Access at http://localhost:8098 (or your configured port).

URLPurpose
/Public mind map / pinboard / tag cloud
/admin.htmlAdmin panel
/screensaver.htmlScreensaver mode
/api/rssRSS/Atom feed
/shared.html?token=...Shared link viewer

Press Ctrl+K or / on desktop. Tap 🔍 on mobile.

Matches: title, category, description, URL, tags. Arrow keys navigate, Enter opens.

Sharing & Friends

Create a Share

  1. Admin → 🔗 Shares → enter name + scope → Create
  2. Copy the token URL
  3. Send to friend (URL + token)

Add a Friend

  1. Admin → 👥 Friends → enter name + their URL + their token → Add
  2. Click Test to verify

Cloudflare Tunnel: Works transparently. The fetch is server-to-server HTTPS — no special config needed.

Android App Mobile

  1. Install Memex-Share-debug.apk
  2. Open → enter server URL + password → Connect
  3. Share any link from any app → "Save to Memex"

Offline queue: if server is unreachable, links save locally and auto-sync when reconnected.

Browser Extension Desktop

Chrome / Edge / Brave

  1. chrome://extensions → Developer mode → Load unpacked → select browser-extension/

Firefox

  1. about:debugging → Load Temporary Add-on → select manifest.json

Click extension icon → Settings → enter URL + password → Connect. Then click on any page to save.

Configuration

Everything in .env:

ADMIN_PASSWORD=your_password
SECRET_KEY=long_random_string
APP_PORT=8098

Change port, restart: docker compose down && docker compose up -d
